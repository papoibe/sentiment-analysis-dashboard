# Sentiment Analysis Dashboard

## Mô tả
Xây dựng dashboard phân tích cảm xúc (sentiment) từ reviews khách hàng trên các nền tảng Google, Facebook. Hệ thống sử dụng AI để tự động phân loại sentiment (Positive/Negative/Neutral) với confidence score, hiển thị biểu đồ trend theo thời gian, và hỗ trợ export báo cáo PDF/Excel. Giúp doanh nghiệp nắm bắt nhanh phản hồi khách hàng, phát hiện các vấn đề tiêu cực kịp thời.

## Công nghệ sử dụng
- Backend: Spring Boot (Java 17+)
- Frontend: ReactJS
- Database: PostgreSQL
- AI: OpenAI API (GPT) - Sentiment Classification, Keyword Extraction
- Security: Spring Security + JWT
- Version Control: Git + GitHub
